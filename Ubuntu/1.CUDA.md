## 安装
### 1.依赖软件  
`sudo apt-get update`  
`sudo apt-get install g++`  
`sudo apt-get install gcc`  
`sudo apt-get install make`  

### 3.卸载之前的显卡驱动
`sudo apt-get remove --purge nvdia* `

### 4.安装显卡驱动
下载地址：[NVIDIA GeForce 驱动程序 - N 卡驱动 | NVIDIA](https://www.nvidia.cn/geforce/drivers/)
